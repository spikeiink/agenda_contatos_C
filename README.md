# Agenda de Contatos em C

Um mini projeto desenvolvido em **C** com o objetivo de praticar manipulação de arquivos, estruturas e funções.  
Permite **adicionar, listar, buscar, alterar e excluir contatos** armazenados em um arquivo de texto.

---

## Objetivo
Este projeto foi criado para fins de estudo da linguagem C, com foco em:
- Manipulação de arquivos (`FILE`)
- Estruturas de dados (`struct`)
- Entrada e saída de dados (`scanf`, `printf`)
- Controle de fluxo e funções

---

## Funcionalidades
- Adicionar novo contato  
- Listar todos os contatos  
- Buscar contato pelo nome  
- Alterar informações de um contato  
- Excluir um contato
